课设重点是：

- 流水、转发、load-use stall、分支 flush
- 后面才是 Cache / 中断 / MMIO

用固定 f₇ 小程序当回归测试负载，和「只输入 n 的通用计算器」是不同目标；硬编码不会扣分，反而指令少、波形好讲、和旧机对照简单。

`Ins[3:0]` 有三种：

| 去向                | 作用                       |
| ----------------- | ------------------------ |
| `RS_addr` → 寄存器堆  | 把这 4 位当作源寄存器编号，读出寄存器里的数据 |
| `Unsigned extend` | 把这 4 位当作无符号立即数，零扩展成 16 位 |
| 直接进入 `EX` 流水线寄存器  | 把地址/编号本身往后级传递（给转发、写回等用   |
