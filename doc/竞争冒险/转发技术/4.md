# 如何在图里画出 MUX 的三个输入

你现在的 MUX 左侧只有 **ID/EX.RS** 一根线，所以看起来像「1 选 1」。要画成 **3 选 1**，需要：

1. **MUX 符号画成 3 个输入**  
2. **另外两根转发线从 EX/MEM、MEM/WB 绕回来**

---

## 1. MUX 符号怎么画

标准画法：**梯形 / 三角形 MUX，左侧 3 个输入，右侧 1 个输出**：

```text
  EX/MEM.alu_result ────────┐
                            ├──┐
  ID/EX.RS ─────────────────┼──┤
                            │  │
  MEM/WB.write_data ────────┼──┤  MUXA  ────> ALU(A)
                            │  │    ↑
                            └──┘  ForwardA
```

Visio / 手绘要点：

| 要素 | 画法 |
|------|------|
| MUX 本体 | 梯形，左侧 **3 条短横线** 表示 3 个输入 |
| 选择信号 | 从下方或侧边进 MUX，标 **ForwardA**（2 bit） |
| 三个输入 | 上、中、下各一根线，**不要合并成一根** |

你现在的图若只有左侧 **1 个输入点**，把 MUX 改成 **3 输入 MUX** 即可。

---

## 2. 三根线分别从哪连、怎么画

### 中间输入（②）—— 已有，保持

```text
ID/EX 寄存器
    │
    ├── RS ──────────────> MUXA 中间输入
    └── RD ──────────────> MUXB 中间输入
```

来自 ID/EX 竖条上的 **RS、RD**，这是「正常读寄存器」路径。

---

### 上面输入（①）—— 从 EX/MEM 绕回来

**来源：** 右边 **EX/MEM** 寄存器里的 **alu_result**

画法：从 EX/MEM 的 result 输出 **向左上方绕一条长反馈线**，接到 MUXA/MUXB 的 **上输入**：

```text
                    ┌──────────────────────────────┐
                    │         EX/MEM               │
                    │   alu_result ────────────────┼──┐
                    └──────────────────────────────┘  │
                                                      │
  ┌───────────────────────────────────────────────────┘
  │
  ├──────────────────> MUXA 上输入
  └──────────────────> MUXB 上输入
```

**技巧：** 反馈线画在 EX 级 **上方**，用虚线或另一种颜色，标 **「转发 EX/MEM」**。

---

### 下面输入（③）—— 从 MEM/WB 绕回来

**来源：** 更右侧 **MEM/WB** 里的 **write_data**（MemToReg 之后）

```text
                    ┌──────────────────────────────┐
                    │         MEM/WB               │
                    │   write_data ────────────────┼──┐
                    └──────────────────────────────┘  │
                                                      │
  ┌───────────────────────────────────────────────────┘
  │
  ├──────────────────> MUXA 下输入
  └──────────────────> MUXB 下输入
```

反馈线画在 EX 级 **下方**，标 **「转发 MEM/WB」**。

---

## 3. 完整 EX 阶段布局（可直接照着画）

```text
  转发 EX/MEM ─────────────────────────────────────────┐
  (alu_result)                                        │
       │                                              │
       ├────────────────────> MUXA上 ──┐              │
       │                              ├──> ALU ──> EX/MEM
       ├────────────────────> MUXB上 ──┘      ↑
       │                                   ALU_src
  ID/EX│                                              │
  ─────┼─ RS ────────────────> MUXA中                │
       │                              ALU(B)        │
       ├─ RD ────────────────> MUXB中 ──> ALU_src    │
       │                              ↑              │
       └─ imm ────────────────────────┘              │
                                                      │
  转发 MEM/WB ─────────────────────────────────────────┘
  (write_data)
       │
       ├────────────────────> MUXA下
       └────────────────────> MUXB下
```

**布局建议：**

```text
        [转发 EX/MEM 走上方]
              ↓   ↓
ID/EX → MUXA → ALU(A)
      → MUXB → ALU_src → ALU(B)
              ↑   ↑
        [转发 MEM/WB 走下方]
```

---

## 4. 你当前图怎么改（三步）

### 第 1 步：换 MUX 符号

把 MUXA、MUXB 从「2 输入 MUX」改成 **3 输入 MUX**（左侧 3 个引脚）。

### 第 2 步：保留中间线

- MUXA 中间 ← **ID/EX.RS**（不变）  
- MUXB 中间 ← **ID/EX.RD**（不变）

### 第 3 步：加两根反馈线

| 新线 | 从哪来 | 接到哪 |
|------|--------|--------|
| 转发线 1 | **EX/MEM.alu_result** | MUXA、MUXB **上输入** |
| 转发线 2 | **MEM/WB.write_data** | MUXA、MUXB **下输入** |

若 MEM/WB 还没画，可先画 **WB 级 MemToReg 输出**，标 `write_data`，再连到 MUX。

---

## 5. 还没画 MEM/WB 时怎么办

第 1 步只有 ADD/ADDI 时，可先画 **占位**：

```text
EX/MEM.alu_result ──> MUX 上   （EX/MEM 已有，可先接）
MEM/WB.write_data ──> MUX 下   （先画虚线框 "MEM/WB"，引脚 write_data）
```

或从 **EX/MEM 再往后画一小段 MEM/WB**，只标 `write_data` 输出即可。

---

## 6. ForwardA / ForwardB 控制线

Forwarding Unit 输出 **ForwardA、ForwardB**（各 2 bit），接到 MUXA、MUXB 的选择端：

```text
forwarding_unit
    ├─ ForwardA[1:0] ──> MUXA.sel
    └─ ForwardB[1:0] ──> MUXB.sel

00 → 选中间 (ID/EX)
01 → 选上面 (EX/MEM)
10 → 选下面 (MEM/WB)
```

图上 MUX 旁可标注：

```text
ForwardA=00/01/10
  00: RS
  01: EX/MEM
  10: MEM/WB
```

---

## 7. 最小可画版本（MUX 旁加注释）

若版面紧，可在 MUX 旁写：

```text
        ┌─ EX/MEM.alu_result
MUXA ───┼─ ID/EX.RS
        └─ MEM/WB.write_data
           (ForwardA 选择)
```

左侧仍画 **3 根线** 进 MUX，不要只画一根。

---

## 一句话

- **中间**：ID/EX 的 RS/RD（你已有）  
- **上面**：从 **EX/MEM.alu_result** 向左绕回  
- **下面**：从 **MEM/WB.write_data** 向左绕回  

把 MUX 改成 **3 输入**，加上面、下面两根 **反馈线**，就是标准的 Forward MUX 画法。
